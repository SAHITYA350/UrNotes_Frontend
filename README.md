# UrNotes - Full Stack Post Management System

UrNotes is a premium, futuristic post management application built with a modern tech stack. It features a sleek "Silk Dark" UI, secure user authentication, and real-time data management.

## 🚀 Tech Stack

### Frontend
- **React 19**: Core framework for building the UI.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS v4**: Modern utility-first CSS framework for styling.
- **Framer Motion**: Premium animations and micro-interactions.
- **Lucide React**: Clean and consistent iconography.
- **React Router Dom**: Client-side routing for seamless navigation.
- **Axios**: Promised-based HTTP client for API requests.
- **React Hot Toast**: Beautiful notification system.
- **React Day Picker**: Interactive calendar components.
- **html2canvas & jsPDF**: Functionality to capture and export content as images or PDFs.

### Backend
- **Node.js & Express**: Fast and minimalist web framework for the API.
- **MongoDB & Mongoose**: NoSQL database for flexible data storage.
- **JWT (JSON Web Tokens)**: Secure stateless authentication.
- **Bcryptjs**: Password hashing for security.
- **Multer**: Middleware for handling file uploads.
- **ImageKit**: Cloud-based image storage and optimization.
- **Dotenv**: Environment variable management.

## ✨ Key Features
- **Premium Silk UI**: A state-of-the-art dark mode design with glassmorphism effects.
- **Authentication**: Secure Login and Sign Up systems with persistent sessions.
- **Post Management**: Create, edit, and share posts with rich media support.
- **Profile Customization**: Personalize user identities within the system.
- **Responsive Design**: Optimized for everything from mobile devices to desktop monitors.

## 🛠️ Getting Started

### Prerequisites
- Node.js installed
- MongoDB URI (Atlas or local)

### Setup
1. Clone the repository.
2. Navigate to `frontend` and run `npm install`.
3. Navigate to `backend` and run `npm install`.
4. Create a `.env` file in the `backend` directory with your configuration.
5. Run `npm start` in the backend and `npm run dev` in the frontend.
